## English

### Source

All sound effects are from [効果音ラボ / Soundeffect-Lab](https://soundeffect-lab.info).  
Files are **not included in this repository** because redistribution is prohibited under their terms of use.

### How to Set Up Sound Files

If you want to build the app with sound, download the relevant files from [soundeffect-lab.info](https://soundeffect-lab.info) and place them in the `Sounds/` folder inside the project.

Downloaded files have Japanese filenames. Rename them to the English names listed below before placing them in `Sounds/`.  
The reason for renaming is to keep source code readable and avoid potential filename encoding issues across environments.

### File List

| Filename | Description |
|---|---|
| `tap.mp3` | Tap / button press |
| `correct.mp3` | Correct answer |
| `wrong.mp3` | Wrong answer |
| `combo.mp3` | Combo milestone |
| `gameover.mp3` | Game over |
| `clear.mp3` | Stage clear / high score |
| `special.mp3` | Special effect / fanfare |
| `unlock.mp3` | Unlock reward |
| `coin_land.mp3` | Coin landing (CoinDrop) |
| `coin_merge.mp3` | Coin merge (CoinDrop) |
| `dollar.mp3` | $1 completed (CoinDrop) |

## 日本語

### 音源について

効果音はすべて [効果音ラボ](https://soundeffect-lab.info) の音源を使用しています。  
利用規約により再配布が禁止されているため、音声ファイルは**このリポジトリには含まれていません**。

### 音声ファイルの準備方法

効果音を有効にしてビルドする場合は、[効果音ラボ](https://soundeffect-lab.info) から該当ファイルをダウンロードし、プロジェクト内の `Sounds/` フォルダに配置してください。

ダウンロード時のファイル名は日本語です。配置する前に、下記の英語ファイル名にリネームしてください。  
リネームの理由は、ソースコード内の参照名を読みやすく保つため、および環境によるファイル名エンコーディングの問題を避けるためです。

### ファイル一覧

| ファイル名 | 用途 |
|---|---|
| `tap.mp3` | タップ・ボタン操作音 |
| `correct.mp3` | 正解 |
| `wrong.mp3` | 不正解 |
| `combo.mp3` | コンボ達成 |
| `gameover.mp3` | ゲームオーバー |
| `clear.mp3` | クリア・好成績 |
| `special.mp3` | 特別演出・ファンファーレ |
| `unlock.mp3` | アンロック・報酬獲得 |
| `coin_land.mp3` | コイン着地（コインドロップ） |
| `coin_merge.mp3` | コイン合体（コインドロップ） |
| `dollar.mp3` | $1完成（コインドロップ） |
